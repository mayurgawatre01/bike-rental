# 🚲 Bike Rental System (Python)

A console-based Bike Rental System built using **Object-Oriented Programming (OOP)** in Python.  
This project simulates a real-world bike rental shop with support for hourly, daily, and weekly rentals, discounts, penalties, and receipt generation.

---

## 📌 Features

- 🚴 Rent bikes on **hourly / daily / weekly** basis  
- 📦 Live stock management  
- 👨‍👩‍👧 Family rental **30% discount** (3–5 bikes)  
- ⏰ Late return **penalty calculation**  
- 🧾 Automatic **receipt generation**  
- ❌ Input validation & error handling  
- 🧠 Clean OOP design (Shop & Customer separation)

---

## 🛠️ Technologies Used

- Python 3.x
- Object-Oriented Programming (OOP)
- CLI (Command Line Interface)
